# Homebrew Tap for Redunce

This is the official Homebrew tap for [redunce](https://github.com/zachsnow/redunce), a tool for finding redundant code using vector similarity.

## Installation

```bash
brew tap zachsnow/redunce
brew install redunce
```

## Updating

```bash
brew update
brew upgrade redunce
```

## More Information

See the [redunce repository](https://github.com/zachsnow/redunce) for documentation and source code.
